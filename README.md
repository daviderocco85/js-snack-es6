# JS Snack es 6

JavaScript object exercises for the web developer course.

# Target

- Snack 1
  Create an array of objects:
  Each object should describe a racing bike with the following properties: name and weight.
  Print result to the console.
 
- Snack 2
  Create an array of football team objects.
  Each team will have several properties: name, points scored, and fouls suffered.
  The name will be the only property to fill in manually; the others will all be initialized to 0.
  Generate random numbers to replace the zeros in the “points scored” and “fouls suffered” properties.
  Finally, create a new array whose elements contain only the team names and the fouls suffered.
  Print everything to the console.

# Solution

The exercise solutions are located in the script.js file, and you can inspect the output through the browser’s DevTools.
